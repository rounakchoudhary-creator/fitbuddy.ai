# FitBuddy AI

FitBuddy is an AI powered fitness planner that generates personalized 7 day workout plans and nutrition tips using Google Gemini AI.

## Tech Stack
- FastAPI
- SQLite
- Gemini AI
- HTML

## Run Project

Install dependencies

pip install -r requirements.txt

Run server

uvicorn main:app --reload

Open browser

http://127.0.0.1:8000
